# comfyui-workflows

Collection de workflows ComfyUI : ImageToImage, ImageToVideo, Inpaint, PNGtoSVG, Pixel_Art, RemBG, Text2Image, Upscale et Favoris.

## Structure

- `Favoris/` : workflows favoris
- `ImageToImage/` : génération image vers image
- `ImageToVideo/` : génération image vers vidéo
- `Inpaint/` : retouche par inpainting
- `PNGtoSVG/` : conversion PNG vers SVG
- `Pixel_Art/` : génération de pixel art
- `RemBG/` : suppression de fond (Remove Background)
- `Text2Image/` : génération texte vers image
- `Upscale/` : amélioration de résolution (upscaling)
